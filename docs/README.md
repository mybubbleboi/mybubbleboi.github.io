>努力研究无聊的重复性的东西是坏事

以我的经验，努力工作从来都不是只上班或“忙于”赚钱或拥有多份工作。有一百万种工作，你根本无法从中学到任何东西。

因此，30多岁的人更要注意，关于倦怠和工作场所界限的警告。如果你没有学到任何东西，不要为无聊的人和事全天候工作。

你可能会因为走得太快而筋疲力尽，因为你不让自己在愚蠢的事项上花费愚蠢的时间来满足你的智力好奇心。

小心这两个结果：培养你对你想坚持的事情的热情。

在各自领域出色的人往往通过非常努力来达到这个地位。如果你能弄清楚一件事，有些只对你上司有利的繁忙工作与让你作为一个人感到自己正在进步并变得更加熟练的工作之间的区别，那么你就准备好学习了。

>人不能没有上进心

>非常努力地工作，非常认真地对待一切
>macwright

之前我就说过：大多数工作不会给你学习的时间许多工作，特别是在技术领域，

没有真正的、有意的教育成分。没有预留的学习时间，没有时间练习，也没有专门的讲师。

你在大学里学到的东西不太可能让你为这份工作做好充分准备。

你可能会有一个很棒的导师，有很多时间，但可能不会。

当你找到一件值得关注的好东西，一件需要投入精力的事情时，它可能是正和。

它可以给你余生的能量，给你一种目标感。人体不像一个能量有限的电池。

你可以做很多事情，比如锻炼、学习和练习，这些都可以奖励和提高你的能力。这是显而易见的，对吗？

如果你有驱动你的东西，而那东西不是工作，也永远不可能是工作，那么当然——得到你能做的最轻的工作。

把时间投入到那个东西上。也许你在工作中所做的是你的主要产出，或产出的一部分，或者只是你为了钱而做的事情。
